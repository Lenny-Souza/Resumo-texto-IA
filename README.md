# Resumo-texto-IA✨
Esse projeto usa AI (inteligência artificial) para criar resumos automáticos de textos.
<br>

# Ferramentas⚙️
- Vite
- React
- Chatgpt
- Puter.js
- Tailwind
<br>

# Funcionalidades✅
- Resumo rápido
- Interface simples
- Suporte a vários idiomas
<br>
