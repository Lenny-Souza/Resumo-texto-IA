# Resumo-texto-IA✨
Esse projeto usa AI (inteligência artificial) para criar resumos automáticos de textos.
<br>
# Funcionalidades✅
- Resumo rápido
- Interface simples
- Suporte a váriios idiomas
<br>
# Ferramentas utilizadas⚙️
- Vite
- React
- Chatgpt
- Puter.js
- Tailwind
